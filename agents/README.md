# Agents

Thư mục này chứa các agent của hệ thống Mina.

Mỗi agent là một thực thể có:
- Vai trò rõ ràng
- Cách phản ứng khi nhận thông tin
- Không chứa suy nghĩ thiết kế, chỉ chứa hành vi

## Template

Để thống nhất format giữa các agent, hãy dựa theo template: `agents/_TEMPLATE.md`.

## Danh sách agent

- `agents/mina_agent.md` — Mina Logic
- `agents/mila_agent.md` — Mila (Discipline / Practical)
- `agents/misa_agent.md` — Misa (Philosophy / Critical Thinking)
- `agents/mita_agent.md` — Mita (Empathy / Emotional-Physical Layer)
